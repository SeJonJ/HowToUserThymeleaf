if : th:if="${조건식}" 을 사용한다

switch : th:switch="${조건변수}" th:case="조건" : 해당 조건에 맞게 태그 내용 출력 th:case="*" : 어떤 조건에도 맞지 않는 경우 => 기타 내용 출력

count username age
username 0
